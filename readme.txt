== Description ==

<p>Import Export WooCommerce Products Orders Subscriptions with Excel and Migrate you eShop Easily with the power of xlsx file. All in One Solution for your WooCommerce data.</p>
<p>This plugin will provide you an interface to easily import or export WooCommerce Simple Products, Orders Import, Orders Export for Simple Products, Coupons, Customers</p>
<p>The plugin works with Ajax with no PHP timeouts and will save you time during Migration </p>
**HPOS WooCommerce Orders Table and PHP 8.2 compatible!**

= HOW DOES IT WORK =

In order to migrate your old eShop to a new website you can follow this series of actions:

- if your old website is in WordPress, install the plugin and Export WooCommerce Products, Export Customers, Export WooCommerce Orders and Export WooCommerce Coupons to Excel.
- Import WooCommerce Products with Excel
- Import WooCommerce Customers with Excel
- Import WooCommerce Coupons with Excel
- Finally do WooCommerce Orders Import with Excel
- Import [WooCommerce Subscriptions in the PRO Version ](https://woocommerce.com/products/woocommerce-subscriptions/ "WooCommerce Subscriptions ") with excel, Export [WooCommerce Subscriptions in the PRO Version ](https://woocommerce.com/products/woocommerce-subscriptions/ "WooCommerce Subscriptions ") with excel

**WOOCOMMERCE IMPORT**

* you go to Import Screen / Import Products, Import Customers, Import Orders or Import Coupons
* there you can download example xlsx file and work based on it
* import the xlsx file
* go to excel columns mapping screen and easily DRAG & DROP the data

**WOOCOMMERCE EXPORT**

* you go to Export Screen / Export Products, Export Customers, Export Orders or Export Coupons
* use criteria for your export query
* choose what fields to export
finally you get a table of fields that can be downloaded in xlsx, csv, or txt format

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Store Migration menu link, 'Store Migration' Submenu on Woocommerce or Settings link on plugins page to use the plugin.

== Frequently Asked Questions ==

= What is the correct sequence of actions to Migrate my WooCommerce eShop ? =
Add WooCommerce Products, Customers, then you can import (if any) Coupons and finally import your WooCommerce Orders.

= Can I upload unlimited products,orders,coupons with this plugin?=
Yes of course, it works with Ajax, there are no PHP timeouts, it imports/updates/exports Simple Products.
